# Blockchain-Based Healthcare Logistics Traceability

A blockchain-powered healthcare logistics management system that ensures secure, transparent, and tamper-proof tracking of medicines throughout the supply chain. The project leverages blockchain technology, smart contracts, and IoT concepts to improve traceability, reduce counterfeit medicines, and enhance trust among manufacturers, distributors, pharmacies, and consumers. ## 📖 Project Overview

Traditional healthcare supply chains often suffer from issues such as:

- Counterfeit medicines
- Lack of transparency
- Data manipulation
- Poor product traceability
- Delayed verification

This project addresses these challenges using blockchain technology by maintaining an immutable ledger of every medicine transaction from manufacturing to delivery.

## 🔄 Workflow

1. Manufacturer registers a medicine.
2. Medicine details are stored on the blockchain.
3. Distributor updates logistics information.
4. Pharmacy receives and verifies the medicine.
5. Customer scans the QR code.
6. Blockchain verifies the authenticity of the medicine.

## 🚀 Installation

```bash

```

Move into the project


